ThermaPace VDOT

Thermal Athletic Performance Engine

Purpose

ThermaPace is a VDOT-based pace calculator that adjusts training paces for real-world weather conditions. It uses a WBGT-inspired Environmental Load Score to model how temperature, humidity, wind, and solar radiation affect your performance, then provides actionable coaching recommendations for each workout type.

How to Use

  1. Enter race performance — input any race distance and time to calculate your VDOT score
  2. Set weather conditions — use the sliders, weather presets, or smart GPS sync to model today's environment
  3. Review recommendations — check the Pacing and Advisory tabs for adjusted paces and training guidance
